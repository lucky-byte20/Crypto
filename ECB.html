ECB (Electronic Codebook) is the simplest mode of AES encryption where each block of plaintext is encrypted independently using the same key.

üîÅ How ECB works?

In ECB mode, the plaintext is divided into fixed-size blocks, and each block is encrypted separately with the AES key, producing identical ciphertext blocks for identical plaintext blocks.
