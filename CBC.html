üîê What is CBC?

CBC (Cipher Block Chaining) is a block cipher mode of AES that encrypts plaintext blocks using both the encryption key and feedback from the previous ciphertext block.

üîÅ How CBC works?

The first plaintext block is XORed with the IV (Initialization Vector) and encrypted; subsequent blocks are XORed with the previous ciphertext block before encryption.
