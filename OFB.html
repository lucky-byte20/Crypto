üîê What is OFB?

OFB (Output Feedback) is a mode of operation for block ciphers that turns them into stream ciphers by using the output of encryption to generate the keystream.

üîÅ How OFB works?

Instead of encrypting plaintext directly, OFB encrypts an IV or the previous output, and XORs the result with the plaintext to generate ciphertext. The encrypted output (not the ciphertext) is used as input for the next step.
