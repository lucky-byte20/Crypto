üîê What is CTR?

CTR (Counter) mode is a block cipher mode that converts a block cipher like AES into a stream cipher by using a counter value combined with the key to encrypt data.

üîÅ How CTR works?

Instead of chaining, CTR mode generates a keystream by encrypting a counter value (usually nonce + block number) with the AES key, then XORs that with the plaintext to produce ciphertext.
